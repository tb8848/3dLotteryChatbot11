包牌游戏说明



包牌游戏说明


  1. 包牌是本站另一种的下注方式，指的是单次下注同类别的多组号码。
    • 口口XX 不能组合口XX口成为一次包牌下注。
    • 二字定不能组合四字定成为一次包牌下注。
    • 只有 "定” 能做包牌下注，"现” 目前不提供包牌下注。
  2. 包牌的赔率独立于散单的赔率，同类别中不同组数的包牌会有其相对应的赔率，显示于各类别的包牌汇总表页面右上方。
  3. 包牌下注中有一个号码中奖，则视为该号码中奖，结算方式同散单下注。中奖请参考规则说明第二章：游戏方法。
  4. 包牌的下注金额代表该次下注中对个别号码的下注金额。例：包牌下注 10 组号码，下注金额输入 1 块，则该次下注总共下注 10 块。


包牌游戏方法


  1. 包牌下注有以下四个入口页面：
    • 二字定
    • 快选
    • txt导入
    • 各类别汇总表 (二字定汇总表、三字定汇总表、四字定汇总表)
  2. 二字定、快选页面的包牌下注：
    1. 先到包牌汇总表查看组数对应的包牌赔率 (请参考包牌游戏说明第 3 点)。例：二字定包牌汇总表：10组-99、15组-98
    2. 选定欲下注的类别。例：口口XX
    3. 选择符合该类别包牌组数的号码数。例：00XX、10XX、20XX、30XX、40XX、50XX、60XX、70XX、80XX、90XX
    4. 输入金额，金额代表对各号码的下注金额。例：1，代表对包牌内每一个号码都下注1块。
    5. 点选 ”包牌" 按钮。若点选 “下注" 按钮则会进行散单下注。
    6. 下注成功
  3. txt导入的包牌下注：
    1. 先到包牌汇总表查看组数对应的包牌赔率 (请参考包牌游戏说明第 3 点)。例：二字定包牌汇总表：10组-99、15组-98
    2. 导入符合该类别包牌组数的档案。例：00XX、10XX、20XX、30XX、40XX、50XX、60XX、70XX、80XX、90XX
    3. 提交确认后点选包牌按钮或是录入汇总表。若点选 “下注" 按钮则会进行散单下注。
    4. 点选包牌按钮：下注成功。
      录入汇总表：参考下述汇总表的包牌下注继续进行下注。
  4. 汇总表的包牌下注：
    1. 查看组数对应的包牌赔率 (请参考包牌游戏说明第 3 点)。例：二字定包牌汇总表：10组-99、15组-98
    2. 依序输入欲下注的号码及金额，直到号码数符合包牌。
    3. 点选包牌按钮。若点选 “下注" 按钮则会进行散单下注。
    4. 确认后下注成功。
