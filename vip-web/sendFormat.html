作业格式

阅读说明：

  1. 系统会自动过滤掉非作业格式中出现的字符
  2. 除单选、和值、和数外，系统会自动删除号码中重复出现的数字字符。
  3. 组三、组六的胆拖下注，系统会自动删除出现在拖码中的胆码数字，例如下注发送：3D组三5拖34567各100，系统识别为：3D组三5拖3467各100。
  4. 系统会自动过滤掉不符合所属类型的无效号码。
  5. 此作业格式仅用于文本快译下注，快选方式下注的文本不适用此作业格式。

通用格式：

  1. 发送"流水"关键字，查看当期累计总投金额。
  2. 发送"盛鱼"关键字，查看剩余积分。
  3. 发送"返"关键字，在开启自助回水后，结算当期回水。
  4. 发送"查"关键字，查看当期作业提交情况。
  5. 发送"盈亏"关键字，查看当期盈亏情况。

上下分格式：

  1. 上分 + 数字，如：上分1000。
  2. 下分 + 数字，如：下分1000。

提交作业格式：

  1. P3/3D + [类型名称] + [号码] + "各" + [金额]

参考示例：

  1. 单选示例：P3/3D + 单选 + 号码1,号码2,号码3,... + '各' + 金额，如: 3D单选123,678,345,897,234,345各10。
  2. 直选示例：P3/3D + 直选 + '百[号码]十[号码]个[号码]' + '各' + 金额，如: 3D直选百123十234个456各100。
  3. 复式包组：P3/3D + 复式 + [号码1,号码2,号码3] + '各' + 金额，如: 3D复式1234各100。
  4. 复式分笔：P3/3D + 分笔复式 + [号码] + '各' + 金额，如: 3D复式1234各100。
  5. 组三包组：P3/3D + 组三 + [号码1,号码2,号码3...] + '各' + 金额，如: 3D组三123456各100。
  6. 组三分笔：P3/3D + 分笔组三 + [号码] + '各' + 金额，如: 3D组三123456各100。
  7. 组三胆拖示例：P3/3D + 组三 + [号码] + '拖' +[号码] + '各' + 金额，如: 3D组三5拖34567各100。
  8. 双飞组三示例：P3/3D + 双飞组三 + [号码] + '各' + 金额，如: 3D双飞组三78各100。
  9. 组六包组示例：P3/3D + 组六 + [号码1,号码2,号码3...] + '各' + 金额，如: 3D组六123456各100。
  10. 组六分笔：P3/3D + 分笔组六 + [号码] + '各' + 金额，如: 3D组六123456各100。
  11. 组六胆拖示例：P3/3D + 组六 + [号码] + '拖' +[号码] + '各' + 金额，如: 3D组六34拖567各100，3D组六4拖567各100。
    12. 如: 3D组六和值7,8,9,10各100。
  12. 双飞组六示例：P3/3D + 双飞组六 + [号码] + '各' + 金额，如: 3D双飞组六78各100。
  13. 和数示例：P3/3D + 和数 + [和值号码](多个用逗号分隔) + '各' + 金额，如: 3D和数7,8,9,10各100。
  14. 跨度示例：P3/3D + 跨度 + [差值号码](多个用逗号分隔) + '各' + 金额，如: 3D跨度7,8,9各100。
  15. 独胆示例：P3/3D + 独胆 + [号码](多个用逗号分隔) + '各' + 金额，如: 3D独胆7,8,9各100。
  16. 1D示例：P3/3D + 1D + '百[号码]十[号码]个[号码]' + '各' + 金额，如: 3D1D百12各100，3D1D个23各100，3D1D百12个23各100。
  17. 2D示例：P3/3D + 2D + '百[号码]十[号码]个[号码]' + '各' + 金额 如: 3D2D百12十23各100，3D2D百12个23各100。
  18. 大小示例：P3/3D + [大小/大/小] + '各' + 金额，如: 3D大各100，3D小各100，3D大小各100。
  19. 奇偶示例：P3/3D + [奇偶/奇/偶] + '各' + 金额，如: 3D奇各100，3D偶各100，3D奇偶各100。
  20. 拖拉机示例：P3/3D + 拖拉机 + '各' + 金额，如: 3D拖拉机各100。
  21. 猜三同示例：P3/3D + 猜三同(或三同号) + '各' + 金额，如: 3D三同号各100。
